Build a mobile-first web app called “Which Indian Prime Minister Are You Most Like?”

Goal:
Create a clean, editorial-style personality quiz web app that feels thoughtful, polished, respectful, and lightly playful. It should work beautifully on mobile devices first, then scale cleanly to tablet and desktop.

Core concept:
This is a fun personality-style quiz based on broad leadership traits, not politics, not ideology, and not historical judgment. The app should match a user to one Indian Prime Minister based on their answers, then optionally show a few other close matches only if the scores are reasonably close.

Important safety framing:
Use respectful language throughout.
Do not mock, insult, caricature, or make defamatory statements about any public figure.
Do not imply that the user literally is a real public figure.
Do not say the result is factual, political, or psychological truth.
Frame everything as a playful personality match based on broad leadership style.
Include clear disclaimers that this is for fun only and does not represent historical judgment, endorsement, or real-life similarity.

Brand and design direction:
Use a clean, minimal, editorial design inspired by modern quiz apps with restrained decoration.
It should feel warm, intelligent, and premium.
Do not make it look overly political.
Do not use the Indian flag, Ashoka Chakra, state emblems, party symbols, or trademarked imagery.
Do not use cliché “exotic India” visuals such as snake charmers, elephants, turbans, etc.
Instead, use subtle India-inspired abstract design accents such as:
- jaali-inspired line patterns
- rangoli-like dot geometry
- soft arch shapes
- tile-border dividers
- petal-like radial motifs
Keep all decorative elements abstract, subtle, low-contrast, and tasteful.

Mobile-first requirements:
This app must be designed for phones first.
Use a single-column layout on mobile.
No horizontal scrolling.
No tiny text.
Buttons and answer cards must be easy to tap with one hand.
Use large readable spacing and strong contrast.
Use these rough breakpoints:
- 0 to 479 px = small phones
- 480 to 767 px = large phones
- 768 to 1023 px = tablet
- 1024+ px = desktop
On desktop, keep the quiz in a narrow centered column so it still feels editorial.

App screens:
1. Landing page
2. Question screen
3. Result screen

Landing page requirements:
Show:
- a small abstract decorative motif at the top
- title: “Which Indian Prime Minister Are You Most Like?”
- subtitle: “18 questions on leadership, values, decisions, and the kind of future you want to build.”
- a short disclaimer
- a primary “Start the Quiz” button
Keep it elegant and minimal.

Question screen requirements:
- one question per screen
- progress bar at top
- question counter
- question text
- four answer cards
- next button
- optional back button
- subtle transition between questions
Do not auto-advance immediately after answer selection unless implemented very smoothly.
Selected answer should highlight clearly.
The “Next” button should activate only after an option is selected.

Result screen requirements:
Show:
- decorative motif
- large result title in the format: “You are Atal Bihari Vajpayee”
- archetype subtitle
- 2 short praise-focused paragraphs
- strengths section
- one gentle growth note
- score card labeled “Your leadership mix” only if other PM matches are reasonably close
- retake button
- share button
- disclaimer at bottom

Score card rule:
Show only one final PM result as the winner.
Only show other PMs in “Your leadership mix” if:
- they are within 3 points of the winner
- and they have at least 6 total points
If no one else qualifies, show only the winning PM and no extra mix card or show just a single-line note.

Data architecture:
Store the quiz content in local JSON.
No database needed.
No auth needed.
Score client-side.
Keep the app lightweight and easy to deploy.
